Getting Started #back to top

This program was designed to help you create your Exhibit A documents faster. You just have to first supply the spreadsheet with all the data in it, then the word document with template fields also. Specify the range of the rows you want to create documents for or select "All Rows" to create documents for all the rows in your spreadsheet.

"File not found" error #back to top

If the file can't be found, try renaming the excel FILE something simpler and restarting this program.

If the error persists

Check the SHEET name is correct, or try renaming the SHEET only (as the sheet name is opened after the file name).

Tips #back to top

Tip #1

Remove spaces or avoid uppercase letters.

Tip #2

Use underscores _ or dashes - instead of spaces.

Tip #3

Copy & paste the file name directly to avoid typing errors.

Dependencies #back to top

Python 3.7.1, lxml, mailmerge, xlrd, python-shell

Python

  • lxml - the most feature-rich and easy-to-use library for processing XML and HTML in the Python language.
  • mailmerge - Performs a Mail Merge on Office Open XML (docx) files. Can be used on any system without having to install Microsoft Office Word
  • xlrd - xlrd is a library for reading data and formatting information from Excel files, whether they are .xls or .xlsx files.

Node.js

  • python-shell - This is used for connecting the app to the python script that handles the conversion
